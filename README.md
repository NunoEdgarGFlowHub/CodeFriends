===========

### Setup

Install mysql and mongodb

```
brew install mysql mongo
```

Install mocha and gulp 

```
npm install -g mocha gulp nodemon
```

Install all packages and bower components

```
npm install
bower install
```

### Running

```
nodemon server
```

### Testing

```
npm test
```
