Collaborate on Code in Real Time

CodeFriends is Google Docs for Sublime Text. Work on code in the same way you always do, but with a toolset that lets your collaborate with anyone around the world.

See When Everyone Else Types

Working real-time on projects means that you can always have the same code your collaborators have, down to the millisecond. Your project will also magically get updated as new files are created and deleted.

Clone Repositories From Git

Use the code you already have by importing it directly from GitHub into a CodeFriends project.

Video and Chat work together remotely

Real Time Communication using WebRTC protocols lets you setup Video Chat with your remote peers without any extra plugins. Persistant chat keeps your important conversations saved so that you won't miss important messages.

Run Your Code CodeFriend compiles

Code with a friend, then run it straight in the browser. Output is logged onto your browser client. CodeFriend compiles 12 different languages: C#, C/C++, Clojure, Java, Go, JavaScript, PHP, Python, Ruby, Scala, VB.NET, Bash, and Objective-C. The syntax highlighter currently supports HTML, CSS, Javascript, PHP, and Ruby. We'll add more soon...
